1. создать новый репозиторий на гитхаб
3. в git bash перейти в нужную дерикторию
2. git init - создать индекс в папке с необходимыми файлами
3. git add - добавить нужный файл
4. git commit -m "комментарий" - создать коммит с комментарием
5. git branch -M main - переименовать ветку, чтобы не называть ниггерским названием
6. git remote add origin https://github.com/Loonaris22/for-ilya.git - к текущему локальному репозиторию добавить удаленную точку
7. git push -u origin main - пушим и привязываем удаленную ветку к локальной (после создания привязки можно будет вводить просто git push)
8. если хотим запушить какие-то изменения - снова добавляем файл, из которого нужно докинуть изменения git add, далее git commit, git push = profit
